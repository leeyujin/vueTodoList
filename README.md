# vueTodoList

- 인프런 'Vue.js 중급 강좌'를 수강하며 정리한 내용입니다
[강의링크](https://www.inflearn.com/course/vue-pwa-vue-js-%EC%A4%91%EA%B8%89)

- 크게 3가지로 구성됩니다
  - cdn을 이용한 TodoApp 구성
  - Vuex를 이용한 리팩토링 (중앙집중화)
  - Helper를 이용한 리팩토링
